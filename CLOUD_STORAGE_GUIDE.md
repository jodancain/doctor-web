# 云存储功能使用指南

## 功能概述

云存储功能展示了微信云开发的云存储能力，包括：

1. **文件上传** - 上传图片、视频等文件到云存储
2. **文件管理** - 查看、预览、删除上传的文件
3. **头像上传** - 用户头像上传到云存储并更新到数据库
4. **CDN加速** - 自动生成CDN访问链接，加速文件访问

## 功能特性

### 1. 文件上传功能

- **支持格式**: 图片、视频等媒体文件
- **上传方式**: 从相册选择或拍照
- **存储路径**: `files/` 目录
- **文件命名**: 时间戳 + 随机字符串，确保唯一性

### 2. 文件管理功能

- **文件列表**: 显示所有上传的文件
- **文件预览**: 点击图片可预览大图
- **文件删除**: 支持删除云存储中的文件
- **文件信息**: 显示文件名、上传时间等信息

### 3. 头像上传功能

- **上传位置**: 用户资料页面，点击头像
- **存储路径**: `avatars/` 目录
- **自动更新**: 上传后自动更新用户数据库记录
- **即时生效**: 上传后立即更新显示

## 使用步骤

### 1. 上传文件

1. 进入"云存储管理"页面
2. 点击"选择文件上传"按钮
3. 选择图片或视频
4. 等待上传完成

### 2. 查看文件

1. 在文件列表中查看所有上传的文件
2. 点击文件可以预览（图片）或查看详情
3. 下拉刷新可以更新文件列表

### 3. 删除文件

1. 在文件列表中，点击文件右上角的删除按钮
2. 确认删除操作
3. 文件将从云存储中删除

### 4. 上传头像

1. 在用户资料页面，点击头像
2. 选择"从相册选择"或"拍照"
3. 选择图片后自动上传
4. 上传成功后头像立即更新

## 技术实现

### 文件上传

```javascript
// 上传文件到云存储
const uploadResult = await wx.cloud.uploadFile({
  cloudPath: 'files/timestamp_random.jpg', // 云存储路径
  filePath: tempFilePath, // 本地临时文件路径
});

// 返回 fileID，可用于访问文件
console.log(uploadResult.fileID);
```

### 文件删除

```javascript
// 删除云存储文件
await wx.cloud.deleteFile({
  fileList: [fileID], // 文件ID数组
});
```

### 文件下载

```javascript
// 下载云存储文件
const result = await wx.cloud.downloadFile({
  fileID: fileID, // 文件ID
});

// 返回临时文件路径
console.log(result.tempFilePath);
```

### 头像更新

```javascript
// 1. 上传头像到云存储
const uploadResult = await wx.cloud.uploadFile({
  cloudPath: `avatars/${userId}_${timestamp}.jpg`,
  filePath: avatarFilePath,
});

// 2. 更新用户数据库记录
await updateUserAvatar(userId, uploadResult.fileID);
```

## 云存储优势

1. **CDN加速** - 文件自动分发到CDN节点，访问速度快
2. **安全存储** - 文件安全存储在云端，不会丢失
3. **自动备份** - 云存储自动备份，数据安全可靠
4. **易于管理** - 通过控制台可以方便地管理文件
5. **成本低廉** - 按实际使用量计费，成本可控

## 存储路径规范

### 文件存储路径

- **用户文件**: `files/{timestamp}_{random}.{ext}`
- **用户头像**: `avatars/{userId}_{timestamp}_{random}.jpg`
- **其他文件**: 根据业务需求自定义路径

### 命名规则

- 使用时间戳确保唯一性
- 使用随机字符串增加安全性
- 保留原始文件扩展名
- 头像统一使用 `.jpg` 格式

## 注意事项

### 1. 文件大小限制

- 单个文件大小限制：根据云开发配置
- 建议压缩图片后再上传
- 视频文件建议使用压缩格式

### 2. 文件类型限制

- 支持常见图片格式：jpg、png、gif、webp等
- 支持视频格式：mp4、mov等
- 其他文件类型根据业务需求

### 3. 存储权限

- 确保云存储权限配置正确
- 测试阶段可以设置为"所有用户可读写"
- 生产环境建议设置合适的权限规则

### 4. 文件列表管理

- 小程序端无法直接列出云存储文件
- 建议使用数据库记录文件信息
- 或者使用云函数获取文件列表

## 常见问题

### 1. 上传失败

**问题**: 文件上传失败

**解决方案**:
- 检查文件大小是否超限
- 检查网络连接
- 检查云存储权限配置
- 查看错误日志

### 2. 文件无法访问

**问题**: 上传的文件无法访问

**解决方案**:
- 检查 fileID 是否正确
- 检查文件是否已删除
- 检查云存储权限设置
- 检查网络连接

### 3. 头像不更新

**问题**: 上传头像后不更新显示

**解决方案**:
- 检查数据库更新是否成功
- 检查用户信息是否已刷新
- 清除缓存后重试
- 检查文件路径是否正确
