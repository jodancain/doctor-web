# 成人肥胖食养指南（2024）知识库精简版

来源：国家卫生健康委办公厅发布《成人肥胖食养指南（2024年版）》。

## 一、判定标准（高频问答）

- 正常体重：18.5 <= BMI < 24.0
- 超重：24.0 <= BMI < 28.0
- 肥胖：BMI >= 28.0
- 中心型肥胖：男性腰围 >= 90cm，女性腰围 >= 85cm

## 二、6 条核心食养原则

1. 控制总能量摄入，保持平衡膳食。
2. 少吃高能量食物，清淡饮食，限制饮酒。
3. 纠正不良饮食行为（定时定量、慢吃、少夜宵）。
4. 多动少静，睡眠充足，作息规律。
5. 结合体质合理使用食药物质。
6. 安全减重，维持长期体重管理。

## 三、可直接用于回答的数字结论

- 常用限能量建议：每日减少约 500-1000 kcal（按个体化评估）。
- 脂肪供能比：20%-30%。
- 蛋白质供能比：15%-20%。
- 碳水供能比：50%-60%。
- 晚餐建议时间：17:00-19:00，餐后不再加餐（可饮水）。
- 每周运动：150-300 分钟中等强度有氧 + 2-3 次抗阻训练。
- 静坐管理：长时间久坐者每小时起身活动 3-5 分钟。
- 睡眠建议：约 7 小时，尽量 23:00 前入睡。
- 减重目标：6 个月减当前体重 5%-10%。
- 合理速度：每月约 2-4kg（避免过快）。

## 四、饮食选择要点

### 推荐
- 全谷物、杂豆、新鲜蔬菜、适量低糖水果。
- 优先低脂高蛋白来源：鱼虾、去皮禽肉、瘦肉。
- 低脂或脱脂奶，不加糖豆制品。

### 需限制
- 高能量密度食物（油炸、甜点、含糖饮料等）。
- 烹调油、盐、添加糖总量。
- 酒精摄入。

## 五、智能体标准提醒语

- 体重管理要“吃、动、睡”三位一体，不能只靠节食。
- 减重过快容易反弹且有健康风险。
- 合并慢病、妊娠哺乳、老年人应由医生或营养师个体化评估。
