# 痛风管家知识库上传包（腾讯元器）

本目录用于提升“痛风管家”智能体在腾讯元器中的检索命中率、回答稳定性与安全性。

## 一、推荐上传顺序（非常重要）

1. `01_珠江医院_就医与预约指引.md`
2. `03_珠江医院_医生门诊时间_结构化版.md`
3. `02_珠江医院_风湿免疫科_能力与技术.md`
4. `04_痛风食养指南2024_知识库精简版.md`
5. `05_肥胖食养指南2024_知识库精简版.md`
6. 原始权威材料（PDF/DOCX）作为补充底座

先上传“短、结构化、强指向”的文档，再上传长文档，可明显减少答非所问。

## 二、文档分工策略

- 短文档（本目录 01-05）负责高频问答和快速命中。
- 长文档（原始 PDF/DOCX）负责权威溯源和细节补充。
- 同类信息只保留一个主版本，避免冲突答案。

## 三、腾讯元器参数建议

- 分段长度：500-800 字
- 分段重叠：80-120 字
- 召回条数：3-5
- 命中阈值：中等偏高（先保守，后按实测调整）

## 四、标签建议（上传时可选）

- `机构信息`
- `就医流程`
- `医生排班`
- `痛风食养`
- `体重管理`
- `权威指南`

## 五、回答边界（建议写入系统提示词）

- 仅做健康科普与就医指引，不替代医生面诊。
- 不提供处方剂量、不指导停药换药。
- 出现急性加重（剧烈疼痛、持续高热、呼吸困难、胸痛等）必须优先建议线下急诊。
- 涉及孕妇、哺乳期、儿童、肾功能不全、多病共存时，默认提示“需医生个体化评估”。

## 六、验收清单（上线前）

- 用 20 个高频问题做抽测（预约、挂号、出诊、饮食禁忌、痛风急性期怎么吃）。
- 目标：命中率 > 85%，且无明显医学风险表达。
- 对错误答案定位来源文档后，做“删冲突文档 or 改文档标题与内容”。
