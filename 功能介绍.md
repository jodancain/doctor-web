# 功能介绍

痛风管家小程序聚焦尿酸与生活方式管理，当前包含以下核心模块：

## 1. 首页健康看板
- 展示最新尿酸值、近 7 天发作次数、饮水总量、运动时长及用药提醒数量。
- 内置快捷入口，跳转至各记录页、食物库、医生列表、健康知识与数据看板。

## 2. 数据记录
- **尿酸记录**：录入数值后自动展示时间线，方便趋势追踪。
- **发作记录**：可选择关节部位、疼痛程度和诱因描述。
- **饮水/运动记录**：输入饮水量与运动类型、时长，形成 7 日统计。
- **饮食记录 / 食物库**：使用颜色标签标记不同嘌呤等级，并可快速记录每日饮食。
- **用药提醒**：记录药品名称、剂量、时间，帮助用户按时服药。

所有记录均通过 `services/records.js` 调用云函数 `healthDataService`，保障参数校验与统一返回格式。

## 3. 数据看板
- 近 7 天的发作次数、饮水量、运动时长曲线，以及饮食颜色分布。
- 云端提前聚合，前端仅负责渲染，提升性能与可维护性。

## 4. 云端能力示例
- `healthDataService` 提供生成小程序码、获取/删除云存储文件的能力示例，可按需扩展。
- `CLOUD_FUNCTION_USAGE.md` 详细说明了每个 `type` 的用途、入参与返回体。

更多产品规划可参考 `PROJECT_OVERVIEW.md` 与 `GOUTCARE_DEV_SUMMARY.md`。
