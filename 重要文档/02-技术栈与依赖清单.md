# 02 — 技术栈与依赖清单

## 一、总体技术栈

```
┌─────────────┬────────────────────────────────────────┐
│    层级      │  技术选型                               │
├─────────────┼────────────────────────────────────────┤
│ 患者端前端   │ 微信小程序原生 (WXML + WXSS + JS)      │
│ 医生端前端   │ React 19 + TypeScript + Vite 6         │
│ 后端框架     │ Express 5 + TypeScript                 │
│ 数据库       │ MongoDB 6.0                            │
│ AI 服务      │ 腾讯元器 (主) / Claude / OpenAI (备)   │
│ 反向代理     │ Nginx                                  │
│ 容器化       │ Docker + Docker Compose                │
│ SSL          │ Let's Encrypt (Certbot)                │
│ 日志         │ Pino                                   │
│ 认证         │ JWT + bcrypt                           │
└─────────────┴────────────────────────────────────────┘
```

## 三、小程序端依赖

小程序为**原生开发**，无 npm 依赖。所有功能均基于：
- 微信小程序基础库（最低要求：2.25.0）
- 微信 API（wx.request, wx.login, wx.getStorage 等）
- 自定义组件（WXML/WXSS/JS）

### 关键配置
```json
// project.config.json
{
  "appid": "wx88328469a06c102e",
  "compileType": "miniprogram",
  "setting": {
    "es6": true,
    "enhance": true,
    "minified": true
  }
}
```
